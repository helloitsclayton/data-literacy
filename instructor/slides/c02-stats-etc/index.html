Data v Stats etc.

Data

Data is a result of the systematic collection and storing of information, usually in machine-readable format.

Most data can be structured as a table (aka tabular), in rows and columns.

Data is often classified as either quantitative (i.e., numerical) or qualitative (non-numerical)

Ask students to think about the intro activity from Class 1 and classify each question (/variable) as quantitative or qualitative

Data

Usually, for tabular data:
Each columns represents a different variable
Each rows represents a different "thing" that the data is describing

Here again the Class 1 activity is a good example, with each question being a variable and each student being described by the data

Statistics

Statistics are summaries of data produced using numerical methods/analysis

Averages, maximums or minimums, frequency/counts

Standard deviation, standard error, quartiles, correlation

Some statistics are pretty basic, some are very complicated and technical! These are all just ways of getting a sense of a dataset in one way or another

Statistics & Data

Doing any sort of analysis of data almost always requires quantitative data.

What does that mean for qualitative data?

Important to point out that working with qualitative data almost always involves some sort of translation into quantitative data. For ex., mapping people's hometowns from Class 1 requires translating a place name into geographic coordinates This even applies to AI chatbots! They're trained using words and phrases translated into mathematical structures called vectors.

Visualizations, Charts, Graphs

Data visualizations are usually based on the elements of a dataset itself

Charts/graphs are drawn from statistical summaries of data

"Data Visualization" is increasingly being used as a blanket term for both
