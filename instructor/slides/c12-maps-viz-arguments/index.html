Maps as Data Visualizations as Arguments

Probably a familiar example even if you haven't been to London, Beck's London Underground map set a trend that we're still following (as evidenced by the next slide) It intentionally abstracted space, location, etc. to make it easier to trace subway lines What sort of arguments does it make?
Here's the current map for the Skytrain transit system in Bangkok, Thailand Follows a lot of the same conventions What arguments does it replicate from Beck's map?
We've looked at the HOLC Redlining maps from Mapping Inequality before, an argument is very foregrounded here (that red is bad and green/blue are good)
Similar map in appearance, but a less obvious argument. This is a Sanborn Fire Insurance Map from the Sanborn Map Company, published 1913 Some arguments:
  • Land is parceled out in the manner indicated
  • Private land ownership is a thing
  • Nothing but buildings matter (for fire insurance)
  • Footprint of buildings are as neat and tidy as depicted
This type of map might or might not be familiar. Clearly privileges highways (those signs are huge!) Do those lines (for roads or rivers) actually correspond to the width of the road or river? No way! City boundaries, do those exist? Only on the map. And look how connected everything is by highways! You should DRIVE EVERYWHERE
